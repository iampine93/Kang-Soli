Service

교육 과정을 수료하는 동안 아래와 같은 작업들을 경험해보았으며, 이 경험들을 실무에서도 잘 보여드릴 수 있습니다.

App Design

사용자 경험을 기반으로 모바일 UI/UX 디자인 및 프로토타입 제작과 최신 트렌드에 맞는 앱 디자인 제작이 가능합니다

Web Design

Grid System을 바탕으로 XD를 활용하여 웹 사이트 디자인 및 프로토타입 제작이 가능하며 웹 사이트의 목적과 컨셉, 사용자의 니즈에 맞는 디자인을 할 수 있습니다.

Web Publishing

HTML5와 CSS3 이용하며 퍼블리싱의 기본기와 활용능력을 갖추고 있고, 상호작용에 필요한 JAVASCRIPT와 JQUERY를 활용하여 웹 페이지의 동적 구현이 가능합니다.

RWD( Responsive Web Design )

Bootstrap과 CSS 프레임워크의 이해를 기반으로 이를 적용 및 활용하여 기기별 적절한 레이아웃을 구성하여 반응형 웹 사이트를 제작할 수 있습니다.

SASS

컴포넌트화 되고 유지보수에 최적화된 코딩을 하기 위해 SASS와 SCSS를 활용할 수 있습니다.

APP Develope

Vue & PWA 를 기반으로 하여 웹앱을 개발한 경험으로 디자인부터 컴포넌트 마크업, 스타일링까지 퍼블리싱 할 수 있습니다.
